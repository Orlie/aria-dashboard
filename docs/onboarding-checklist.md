**Start Date:** [START_DATE]

---

## Day 0 — Contract Signed ✍️

- [ ] Signed agreement received
- [ ] First month payment confirmed (Wise/PayPal)
- [ ] Send "Onboarding Welcome" message (use template in ARIA)
- [ ] Add client to ARIA Dashboard (Revenue → Add Client)
- [ ] Create client Slack/Discord channel (internal)

## Day 1 — Access & Information Gathering 🔑

- [ ] Request TikTok Shop Seller Center collaborator access
- [ ] Request TikTok Ads Manager access (if running ads)
- [ ] Receive product catalog (minimum top 10 SKUs)
- [ ] Receive brand guidelines (colors, tone of voice, do's/don'ts)
- [ ] Receive product pricing + margin information
- [ ] Get list of top 5 competitors
- [ ] Document client's goals and success metrics

## Day 2–3 — Strategy & Setup 📋

- [ ] Audit current TikTok Shop (listings, reviews, content)
- [ ] Run KaloData analysis on client + competitors
- [ ] Set up product knowledge document for LIVE hosts
- [ ] Create initial LIVE selling schedule (frequency, times)
- [ ] Set up GMV Max ad campaigns (if applicable)
- [ ] Create creator recruitment brief
- [ ] Draft first week's LIVE scripts
- [ ] Set up Discord server for creator community

## Day 4–5 — Creator Program Launch 🎬

- [ ] Identify 20–30 potential creators (TikTok search + affiliate marketplace)
- [ ] Send first batch of creator recruitment DMs (use template in ARIA)
- [ ] Coordinate free sample shipment logistics with client
- [ ] Set up affiliate commission rates in TikTok Shop
- [ ] Create creator onboarding document

## Day 5–6 — Test & Train 🎯

- [ ] LIVE host training session on client's products
- [ ] Test LIVE stream (private or with small audience)
- [ ] Review test stream with team
- [ ] Adjust scripts and product presentation order
- [ ] Confirm LIVE schedule with client

## Day 7 — GO LIVE 🔴

- [ ] First official LIVE selling session
- [ ] Monitor chat engagement, conversion, and technical issues
- [ ] Post-LIVE performance report sent to client
- [ ] Debrief with host — what worked, what to improve

## Week 2 — Optimize 📈

- [ ] First weekly performance report sent
- [ ] Adjust LIVE schedule based on Day 7 performance
- [ ] Follow up with creators who received samples
- [ ] Optimize product listings based on LIVE selling insights
- [ ] Review ad performance and adjust bids
- [ ] 15-min weekly check-in call with client

## Ongoing — Monthly Rhythm 🔁

- [ ] Weekly performance reports (every Monday)
- [ ] Monthly strategy review call with client
- [ ] Monthly invoicing (1st of each month)
- [ ] Creator program expansion (recruit 5–10 new creators/month)
- [ ] LIVE schedule optimization based on data
- [ ] Update ARIA Dashboard with monthly revenue data

## Red Flags to Watch For

- Client not responding within 48 hours → Escalate via text/call
- No sample products shipped after 5 days → Pause creator recruitment
- LIVE session viewership below 10 → Adjust timing, add ads, change host
- Creator not posting within 7 days of sample → Follow-up DM
- Client disputing invoice → Reference contract Section 2
